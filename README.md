# AI-BOT-DOTA 🤖🎮

Система создания, обучения и интеграции умных ИИ-ботов для **Dota 2**. Проект включает в себя парсинг профессиональных матчей (демок), обработку данных на Python, обучение моделей машинного обучения для различных позиций (с 1 по 5) и экспорт весов в Lua-скрипты, которые выполняются прямо внутри движка Dota 2.

В комплекте также идет удобный дашборд для мониторинга и кастомизации игрового стиля ботов.

---

## 🛠️ Архитектура и Структура проекта

Проект разделен на логические блоки: сбор данных, обучение ИИ, интеграция с игрой и аналитика.

```text
├── parser/                      # Парсер демок (на Go) для извлечения игровых логов
├── model/                       # Сохраненные обученные модели (.joblib) для всех позиций
├── data/                        # [Игнорируется Git] Локальные дампы, логи и сырые данные матчей
│
├── ai_coach.py                  # Модуль "тренера" для подсказок и динамической оценки
├── clean_and_train.py           # Скрипт очистки данных и запуска базового обучения
├── trainer.py                   # Основной пайплайн обучения ML-моделей
├── export_model_to_lua.py       # Конвертер весов моделей в Lua-таблицы для Dota 2
│
├── bot_generic.lua              # Базовая логика ИИ-бота в игре
├── hero_selection.lua           # Алгоритм выбора героев (драфт)
├── item_purchase_generic.lua    # Динамический закуп предметов на основе меты
├── ability_item_usage_generic.lua # Логика прожатия скиллов и итемов
│
├── dashboard.py                 # Backend веб-интерфейса управления ботами
├── DotaBotDashboard.exe         # Скомпилированный GUI-клиент для мониторинга
├── server.py                    # Локальный API-сервер для связи игры и Python
│
├── continuous_monitor.py        # Скрипты автоматизации и непрерывного
└── pipeline_runner.py           # конвейера (сбора данных -> переобучения)
